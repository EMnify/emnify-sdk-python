Getting Started#

Install#

Requirements#

Python > 3.6 is required.

Install From Source Code#

Installation commands#
git clone https://github.com/emnify-python-sdk.git
cd emnify-python-sdk
python setup.py install

Install via PyPi#

Installation commands#
pip install emnify-python-sdk

Configure#

Create Application Token#

In order to use SDK you gonna need to create your Application Token via: Enterprise Portal(Simplest Way) or API or , and apply it to initiate SDK.

Use in Code#

Example of Usage#
 1TOKEN = '<PASTE YOUR APPLICATION TOKEN HERE>'
 2
 3# Import package
 4from emnify import EMnify
 5
 6# Initiate SDK instance using application token
 7
 8emnify = EMnify(TOKEN)
 9
10# Execute a command against desired API
11devices = emnify.devices.get_devices_list()
